# 🌱 GreenLoop Management System

GreenLoop Management System is a modern, eco-friendly operations platform designed to streamline business processes including staff management, inventory tracking, order handling, supplier coordination, and delivery management.

---

## 🚀 Features

* 👤 **User & Role Management**
* 📦 **Inventory Tracking**
* 🛒 **Order & Supplier Management**
* 🚚 **Delivery Assignments**
* 📊 **Dashboard Overview**
* 🔐 **Secure Login System**
* ⚙️ **User Profile & Settings (Enhancing)**

---

## ⚙️ Setup Instructions

1. Open MySQL Workbench
2. Import and execute `GreenLoop_db.sql`
3. Navigate to project directory:

   ```bash
   cd GreenLoopManagementSystem/GreenLoopSystem
   ```
4. Run build script:

   ```bash
   build.bat
   ```
5. Run application:

   ```bash
   run.bat
   ```
